SSH use pub/private key for authentication. And key is built from certain account. Basically, we distribute pub keys to all servers that we can use one account to all of them

ssh-keygen -t rsa
scp ~/.ssh/id_rsa.pub hadoop-user@target:~/master_key
mkdir ~/.ssh
chmod 700 ~/.ssh
mv ~/master_key ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
If you want to enable auto login locally, you need to copy id_rsa to /.ssh/ too.
